============================================================
A. RINGKASAN UTAMA YANG WAJIB DIPAHAMI
============================================================

AntiRibet adalah platform digital all-in-one untuk membantu berbagai jenis bisnis membuat mini website, mengelola katalog produk atau layanan, menerima transaksi, menggunakan sistem kasir/POS, mengatur QR order, booking, antrean, invoice, customer database, staff management, laporan penjualan, dan saldo transaksi dalam satu ekosistem.

AntiRibet harus dipahami sebagai:
**Business Operating Platform / Business Operating System**

Model bisnis utama:
Merchant top up saldo.
Saldo merchant dipotong Rp500 untuk setiap transaksi berhasil.
Tidak ada biaya bulanan wajib.
Tidak ada premium domain.
Tidak ada premium subdomain.
Tidak ada biaya per produk.
Tidak ada biaya per item.
Fee Rp500 adalah per transaksi berhasil.

Untuk MVP:
Customer membayar langsung ke merchant.
AntiRibet tidak memegang uang penjualan customer.
Merchant membayar AntiRibet melalui saldo deposit.
AntiRibet mendapatkan revenue saat saldo merchant dipotong Rp500 per transaksi berhasil.

Tech stack final:
Frontend: Flutter untuk Web, Android, dan iOS.
Backend: Laravel API.
Database: PostgreSQL.
Queue / Cache: Redis.
Realtime: Laravel Reverb / WebSocket.
Auth: Laravel Sanctum API Token.
